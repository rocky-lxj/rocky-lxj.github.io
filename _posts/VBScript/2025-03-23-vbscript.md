## vb 简介
VBScript 是一种脚本语言
脚本语言是一种轻量级的编程语言
VBScript 是微软的编程语言 Visual Basic 的轻量级的版本

## 语法
option Explicit 是为了用来告诉系统 必须规范 必须用dim声明 
dim 变量名
``` vbs
option Explicit
dim first,second
first = CINT(InputBox("enter one ","first"))
second = CINT(InputBox("enter second"&vbcrlf& "to sum","second",0))
` &vbcrlf& 自行换行
`第一个空代表框名
`第二个空代表框中出现的文字
`第三个代表初始化的值
dim sum
sum = first + second
MsgBox "the sum is" &sum
```
